# login-registraion-system-passportjs-nodemailer

This is an user login and registration system api, Which include api such as login, registration, and forget password

## Installation

Download the zip file or clone the project from github

## To Install run
```bash
npm install
```


## Usage
To use, check the respective route file and app.js file (base file)

### To register
localhost:3000/api/register

### To login
localhost:3000/api/login
